
# Foo-thread Library

## Setup (In Linux Systems only) 

1. Clone the repository
     ```bash
      git clone https://github.com/CHERUKURITAPASWINI/Foo-thread-Library.git
      cd Foothread_Library_Operating_Systems/
    ```
     
2. Run the Code
   ```bash
      make newrun
    ```
   
3. Clean
   ```bash
      make clean
    ```
